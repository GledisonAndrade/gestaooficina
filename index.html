Sobre o Sistema

Este sistema foi desenvolvido para gerenciar ordens de serviço e garantias em oficinas mecânicas.

Objetivo: Facilitar o controle de serviços realizados, garantias e geração de relatórios para o setor de pós-vendas.

Desenvolvedor: Gledison Arruda Andrade

Funcionalidades:

  • Abertura e fechamento de ordens de serviço
  • Gestão de garantias com registro fotográfico
  • Relatórios e gráficos de desempenho
  • Exportação de dados para PDF
  • Interface responsiva para desktop e mobile
  • Modo claro e escuro
